# Nexus Messenger v2.0

## Запуск через CMD (Windows)

### 1. Установите Node.js
Скачайте и установите с https://nodejs.org (версия 18 или выше)

### 2. Распакуйте архив
Распакуйте в любую папку, например `C:\nexus`

### 3. Откройте CMD в папке проекта
```
cd C:\nexus
```

### 4. Установите зависимости (только в первый раз)
```
npm install
```

### 5. Запустите сервер
```
node server.js
```

### 6. Откройте в браузере
```
http://localhost:3000
```

---

## Остановка сервера
Нажмите `Ctrl+C` в окне CMD

## Смена порта
```
set PORT=8080
node server.js
```
