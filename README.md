# Nexus Messenger v2.0

## Запуск через CMD (Windows)

### Шаг 1 — Установите Node.js
Скачайте и установите Node.js версии 18+ с https://nodejs.org

### Шаг 2 — Откройте папку проекта в CMD
```
cd C:\путь\к\папке\nexus-final
```

### Шаг 3 — Установите зависимости (только первый раз!)
```
npm install
```

### Шаг 4 — Запустите сервер
```
node server.js
```

### Шаг 5 — Откройте браузер
```
http://localhost:3000
```

---

## Остановить сервер
Нажмите `Ctrl+C` в окне CMD

## Сменить порт
```
set PORT=8080
node server.js
```

---

## Деплой на Railway
1. Загрузите файлы в GitHub репозиторий
2. Подключите репозиторий к Railway
3. Railway автоматически запустит `npm start`
